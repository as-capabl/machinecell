machinecell
===========

Arrow based Stream transducer.

Features
---------------
* Monadic (ArrowApply) side effect
* 1-to-N output
* Coroutine (Plan monad)
* AFRP-like utilities (hold, edge, switch...)

Status
---------------
Experimental.
This version has performance issue.
And APIs not fixed.
